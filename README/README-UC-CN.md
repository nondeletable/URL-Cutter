## 🔗 此应用的功能

- 通过几次点击即可将长链接转换为短链接。
- 一键复制生成的短链接。
- 保存可搜索的链接历史记录（避免链接丢失）。
- 支持按 URL / 服务 / 日期筛选，并可自定义表格列。
- 非常适合在聊天、社交媒体、文档或评论中分享链接。
&nbsp;
&nbsp;

## 🎨 功能特点

- 即时缩短长 URL。
- 通过 `pyshorteners` 使用 TinyURL 服务。
- 一键复制到剪贴板。
- 历史记录窗口（表格视图）。
- 响应式且极简的界面设计。
&nbsp;
&nbsp;

## 😎 隐私与安全

- ✅ 无账号、无分析、无遥测。
- ✅ 历史记录仅存储在本地设备中（SQLite + SQLAlchemy）。
- ⚠️ 在生成短链接时，URL 会被发送至 TinyURL 服务（这是缩短链接的工作方式）。
&nbsp;
&nbsp;

## ⚒ 安装

- 前往 **Releases** 下载最新的 ".exe" 文件。
- 解压到任意位置（桌面、工具文件夹、USB 设备等）。
- 运行 "URL Cutter.exe"。
&nbsp;
&nbsp;

## 🏓 使用方法

1. 粘贴一个长 URL。
2. 点击 **CUT**。
3. 点击 **Copy** 图标以复制短链接。
4. 打开 **History**，以便后续搜索、筛选和重复使用链接。

## 💾 技术栈

- Python 3.13
- Flet (UI)
- pyshorteners (TinyURL)
- SQLAlchemy + Alembic（历史记录存储与迁移）
- pytest + coverage（测试）
- ruff, black, pre-commit（代码质量）
- PyInstaller（Windows 构建）
